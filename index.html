Week 12: RestDB

I got introduced to Rest API, RestDB and NoSql:

  • APIs and how to use them
  • Calling APIs: GET, POST, PUT, DELETE
  • Status Codes e.g. 200 - data was received and operation was performed
  • AJAX & Postman
  • Differences between SQL & NoSQL and when to use each one:
    SQL:
    - Best fit for data with tables and rows
    - Works better for small datasets
    - Frequent updates
    - Strong dependency on multi-row transactions
    - Modify large volume of records
    NoSQL:
    - Best fit for unstructured data
    - Works better for large datasets
    - High write loads
    - High availability in unstable environment
    - Data is location based
